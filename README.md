# COVID-19 Early Case Trend Analysis & Recovery Insights

## Overview

This project was completed as part of the Artificial Intelligence Minor Project under the Ediglobe Internship Program.

The objective of this project is to analyze COVID-19 patient data to understand infection patterns, recovery trends, and factors influencing recovery duration using Python and data analytics techniques.

## Project Objectives

* Analyze demographic patterns such as age, gender, country, and region.
* Study infection spread using infection reasons and contact exposure levels.
* Evaluate recovery trends based on confirmation and release dates.
* Identify the most impacted regions.
* Predict recovery duration using Linear Regression.

## Dataset Features

The dataset contains the following information:

* Demographics: sex, birth_year, country, region
* Infection Details: infection_reason, infection_order, infected_by
* Exposure Metrics: contact_number
* Timeline Data: confirmed_date, released_date, deceased_date
* Case Outcome: state

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Analysis Performed

* Exploratory Data Analysis (EDA)
* Descriptive Statistics
* Data Visualization
* Correlation Analysis
* Linear Regression
* Model Evaluation using R² Score and RMSE
* Residual Analysis

## Key Findings

* Identified demographic trends among infected patients.
* Analyzed infection spread patterns.
* Studied recovery duration and regional impacts.
* Observed weak relationships between selected features and recovery time.
* Built a Linear Regression model for recovery prediction.
